How do you configure which sets go on which page?

This is the core feature — pick the approach that feels most natural to use.

Module Settings
Page 1 Slots 1, 4, 5
Page 2 Slots 2, 3, 6
Page 3 Slots rest

A — Text input in module settings

One TextSetting per page in the Cobalt settings panel. Type slot numbers separated by commas. Slots not assigned go to a default "overflow" page.

B — Edit mode inside the wardrobe GUI

An "Edit" button in the wardrobe overlay. Each set shows a page dropdown. Click a set to cycle/pick which page it lives on. Changes save instantly.

C — Dedicated page builder panel

A separate Cobalt panel (like the route picker) with columns per page. Drag sets between columns to assign them. Most visual but most work to build.
