Tech Stack

What framework should both panels be built with?

A

Next.js (React)

Most popular choice for dashboards. SSR + API routes built-in. Huge ecosystem of UI libraries (shadcn/ui, Radix, etc.).

  • Best component library ecosystem
  • Easy to deploy (Vercel, static, Node)
  • Heavier bundle
B

SvelteKit

Lighter, faster, and arguably simpler. SSR + routing built-in. Less boilerplate than React.

  • Smallest bundle size
  • Very clean syntax
  • Smaller ecosystem vs React
C

Vite + React (SPA)

Simple static SPA — no SSR, no server. Just a React app bundled with Vite. Served as static files from the VPS.

  • Simplest deployment (just static files)
  • No Node server needed
  • No SEO, slightly slower first load
