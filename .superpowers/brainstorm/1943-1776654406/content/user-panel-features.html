User Panel — What pages does it need?

Select everything the user-facing panel should include

1

Login / Register

Sign in with Cobalt credentials. First-time users create an account.

2

License Key Redemption

Enter a license key to activate their subscription. Maps to POST /enroll/redeem.

3

Account Dashboard

Overview of subscription status, plan tier, expiry date, and bound device info.

4

Device / Binding Info

Show current binding status, HWID status, Minecraft username. Contact support to reset.

5

Download Page

Links to download the bootstrapper and get started. Shown after login/redemption.
